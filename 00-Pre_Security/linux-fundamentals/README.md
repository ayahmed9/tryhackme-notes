# Linux Fundamentals Notes

- `pwd` – show current directory
- `ls` – list files
- `cd` – change directory
- `cat` – view file contents
- `chmod` – change permissions
- `chown` – change file owner
- `sudo` – run as superuser
- `touch` – create empty file
- `rm` – delete file
- `mv` – move or rename
- `cp` – copy files
